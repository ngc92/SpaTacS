# SpaTacS
space tactics simulator

controls:
  * scrolling WASD + QY updown
  * left click: select ship
  * right click: move to / attack
  * mouse wheel: change target height
  * Shift click: set waypoints
  * weapon modes:
    - 1 armour piercing
    - 2 high explosive
    - 3 shield overload
  * pause: P
